# p5.js 学习

[查看英文](./README.md)

这个仓库专门于分享我的 p5.js 作品和任何关于 p5.js 的知识。

这个仓库包含很多例子，比如以下几个大类。

- 植物、风景编码的思路
- 文本效果
- 几何图案
- 循环动画
- 音乐可视化
- 生成艺术
- 自然系统的模拟
- 机器学习 API
- 数学
- 其他

本仓库涵盖了非常广泛的话题供您参考。如果你打算开始学习 p5.js，你就来对地方了。
